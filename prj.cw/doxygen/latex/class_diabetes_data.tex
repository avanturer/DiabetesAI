\doxysubsubsection{Подробное описание}
Класс \doxyref{Diabetes\+Data}{стр.}{class_diabetes_data} предназначен для подготовки датасета 

Класс \doxyref{Diabetes\+Data}{стр.}{class_diabetes_data} предоставляет возможность подготовить датасет для дальнейшей работы с классом \doxyref{Logistic\+Regression}{стр.}{class_logistic_regression} и \doxyref{Plot}{стр.}{class_plot}, путем разбиения датасета на двумерный вектор double и одномерный вектор int\+: вектор X\+\_\+ содержит в себе строки датасета, не включая названия столбцов и исход(outcome), и у\+\_\+, содержит в себе лишь исход(outcome)
